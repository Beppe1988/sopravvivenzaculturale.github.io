# sopravvivenzaculturale.github.io
*Sopravvivenza Culturale* è uno spazio di analisi dedicato ai rapporti tra cultura, società, geopolitica, tecnologia e potere. Osserva le trasformazioni del presente, le crisi dell’Occidente, il ruolo dei media, l’impatto della tecnica e la sopravvivenza dei simboli, della memoria e del pensiero critico nell’età digitale.
